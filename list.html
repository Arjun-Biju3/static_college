HTML Lists

Ordered Lists (ol)

An ordered list is a numbered list where each item is marked with a number or other ordered marker.

  1. This is the first item in an ordered list.
  2. This is the second item.
  3. And this is the third item.

Unordered Lists (ul)

An unordered list is a bulleted list where each item is marked with a bullet point or other unordered marker.

  • This is the first item in an unordered list.
  • This is the second item.
  • And this is the third item.

Definition Lists (dl)

A definition list consists of a list of terms and their definitions.

Term 1
Definition of Term 1.
Term 2
Definition of Term 2.
